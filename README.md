# WaniKani Statistics Tracker

Advanced statistics and analytics PWA for WaniKani with offline support, leech detection, and predictive insights.

## Features

- 📊 Comprehensive statistics dashboard
- 🔍 Advanced leech detection with confusion pair analysis
- 📈 Predictive analytics and completion forecasting
- 📱 Mobile-first PWA with offline support
- 🎯 Goal setting and achievement tracking
- 🌙 Dark mode support

## Setup

1. Get your WaniKani API token from https://www.wanikani.com/settings/personal_access_tokens
2. Open the app
3. Enter your API token when prompted
4. View your statistics!

## Development
```bash
# Install dependencies
npm install

# Run local dev server
npm start

# Build for production
npm run build
```

## Privacy

Your API token is stored locally in your browser and never transmitted to any third-party servers. All data processing happens client-side.
